# 📊 YouTube Trending Analytics Dashboard (Power BI)
![Youtube](https://wallpapers.com/images/featured/youtube-background-34ycn949dz42yg4c.jpg)
## 🚀 Project Overview
This project presents an **interactive Power BI dashboard** analyzing YouTube trending videos across different countries, categories, and channels.  
The goal is to uncover **content performance insights, audience engagement patterns, and global trends** using data-driven techniques.

---

## 🎯 Objectives
- Analyze **trending video performance**
- Identify **top-performing categories and channels**
- Understand **viewer engagement (likes, comments)**
- Explore **geographic distribution of trends**
- Track **trending patterns over time**

---

## 🛠️ Tools & Technologies
- 📊 Power BI (Data Visualization)
- 🧹 Power Query (Data Cleaning)
- 📐 DAX (Data Analysis Expressions)
- 📁 CSV Dataset

---

## 📂 Dataset Information
The dataset includes the following key fields:
- 🎬 Video Title & Channel Name
- 🌍 Country
- 📅 Publish Date & Trending Date
- 👀 Views
- 👍 Likes
- 💬 Comments
- 📌 Category

## 📈 Key Insights
- 📌 Certain categories consistently dominate trending content  
- 📌 A few channels contribute to a large share of total views  
- 📌 Engagement varies significantly across videos with similar views  
- 📌 Some videos trend immediately after publishing (viral content)  
- 📌 Content preferences differ across countries
